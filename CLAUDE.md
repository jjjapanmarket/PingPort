# CLAUDE.md — PingPort

日本人卓球選手と海外クラブチームをつなぐアプリ。開発者本人の海外プレー経験に基づく個人開発。

## 最重要ルール: Claude はコードを書かない

このプロジェクトは「理解のための 0→1」が目的。

- **Claude は Edit / Write ツールでソースコードを作成・編集しない。** 差分をコードブロックで「何を・どこに・なぜ」の形で示し、開発者本人が Android Studio で手を動かす。
- 例外は、本人が「書いて」と明示的に頼んだときだけ。
- 編集後の確認、エラーの原因解説、コミットの区切り提案は積極的にやってよい。

## 進め方のスタイル

- 説明はすべて日本語。初学者向けに、専門用語は初出時にかみ砕く。
- 差分は1箇所ずつ。コピペで丸ごと置き換えさせない。
- 「Kotlin文法は深く理解する / UI装飾オプションは深追いしない」の仕分けで進める。
- 区切りごとにコミット＆push し、GitHub の diff で振り返る（差分学習）。

## 技術方針（MVP = v1）

- Kotlin / Jetpack Compose
- 対象ユーザーは「海外チームを探す日本人選手」のみ。画面は一覧と詳細の2つ。
- v1 でやらない: ログイン、投稿、メッセージ、検索・絞り込み、サーバー（データはダミーをベタ書き）
- 画面遷移は Navigation ライブラリを使わず `remember + mutableStateOf` の状態切り替え（状態管理の学習優先。Navigation へのリファクタは v2 以降）
- v2 の構想: ViewModel + Repository + 単体テスト + 実データ化（詳細は Notion のロードマップが正）
